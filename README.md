# ShopIt



Shop it is a mock online store for practicing react and Redux toolkit

## Built With

- Typescript
- React
- Noroff API
- CSS Modules

## Getting Started

### Installing

1. Clone the repo:

```bash
https://github.com/SiriHoyas/frontend-frameworks-ca-SiriHoyas
```

2. Run NPM install to install dev dependencies:

```bash
npm install
```

3. Build project with Vite

```bash
npm run build
```

4. Run the site

```bash
npm run preview
```
